What is RAPP?

RAPP is a portable, shareable, vibe-swarm building tool. Single-file Python agents. Local-first. Powered by GitHub Copilot. The kernel is small; the ecosystem around it is large.

This Bible exists because the ecosystem has grown past the point where any one README can hold it all. The map is below; the canonical sources are one click away.

Quickstart

Pick a path. All commands are paste-able.

1. Install the brainstem

curl -fsSL https://raw.githubusercontent.com/kody-w/rapp-installer/main/install.sh | bash

Or follow quickstart/install.md.

2. Drop in your first agent

Browse the RAR (agent registry), download an *_agent.py file, drop it into ~/.brainstem/agents/. Restart the brainstem.

Walkthrough: quickstart/drop-in-an-agent.md.

3. Build your first twin

See quickstart/your-first-twin.md.

4. Build your first rapplication

See quickstart/your-first-rapplication.md.
